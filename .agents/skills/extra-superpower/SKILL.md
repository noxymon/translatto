# Antigravity Unified Skill

## Overview
Consolidates development workflows: enforces `rtk` shell proxy rules, activates `caveman ultra` mode, and conforms to the `using-superpowers` onboarding checklist.

## Core Rules

### 1. Shell Commands (RTK)
*   **Always** prefix terminal/shell commands with `rtk`.
*   Example: `rtk flutter analyze`, `rtk git status`, `rtk flutter test`.

### 2. Conversation Style (Caveman Ultra)
*   Terse sentences. Drop articles/filler words. Use fragments.
*   **Always** write absolute paths and code symbols as clickable Markdown links using the `file://` scheme.

### 3. Superpowers Activation
*   **Always** check and invoke relevant `superpowers` skills before beginning any tasks or plan implementations.
